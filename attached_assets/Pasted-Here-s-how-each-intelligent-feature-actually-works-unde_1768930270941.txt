2. CASTING CALL SCANNING (Real AI Web Scraping)
Uses Core.InvokeLLM with internet access:

const response = await base44.integrations.Core.InvokeLLM({
  prompt: `Search Google and these URLs: ${sources.map(s => s.url).join(', ')}
  
  Find current UGC casting calls, brand collaborations, and sponsorship opportunities 
  specifically for cat influencers and pet content creators.
  
  Return JSON with opportunities array containing: title, brand_name, description, 
  requirements, compensation_type, deadline, platforms, contact_email, source_url`,
  
  add_context_from_internet: true,  // THIS IS KEY - gives LLM web search access
  
  response_json_schema: {
    type: "object",
    properties: {
      opportunities: { type: "array", items: { ... } }
    }
  }
});

// Save results to database
await base44.entities.CastingCall.bulkCreate(response.opportunities);
How it works:

add_context_from_internet: true = LLM gets real-time Google search, Google Maps, news APIs
LLM searches the web for "cat UGC casting calls", "pet brand collaborations", etc.
Extracts structured data from websites, forums, job boards
Returns clean JSON that gets saved to your database

WHY IT'S REAL, NOT FAKE:
Real Database - Base44 entities persist data. When you create a Project, it's stored permanently.

Real AI - Core.InvokeLLM uses actual LLMs (Claude/GPT-class models) with internet access.

Real Web Data - add_context_from_internet: true means the AI searches Google, scrapes websites, reads news in real-time.

User-Generated Content - Most data comes from users manually adding projects, or pasting emails for AI extraction.

No Mock APIs - There are no setTimeout() fakes or hardcoded JSON responses. Everything hits real APIs.

THE LIMITATION:
It doesn't connect to TikTok/Instagram APIs directly because those require OAuth and are platform-specific. Instead:

User manually enters their follower counts
AI searches the PUBLIC web for trends and opportunities
User uploads their own project data
To make it FULLY automated, you'd need:

TikTok/Instagram OAuth to auto-fetch follower counts
Backend cron jobs to run daily scans automatically
Webhooks to monitor casting call websites 24/7
But the current implementation gives users AI-powered tools that do real work - just triggered manually instead of running on autopilot.
